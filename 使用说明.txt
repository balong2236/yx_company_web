和合堂中医养生馆官网使用说明

欢迎使用和合堂中医养生馆官网模板！这是一个纯静态的HTML网站，可以直接上传到GitHub Pages托管。以下是使用说明：

一、文件结构

├── index.html          # 首页
├── css/                # 样式文件夹
│   └── style.css       # 主样式文件
├── images/             # 图片文件夹（预留）
├── posts/              # 博客文章文件夹
│   ├── article-template.html  # 博客文章模板
│   ├── article1.html   # 示例文章1
│   ├── article2.html   # 示例文章2
│   └── article3.html   # 示例文章3
└── 使用说明.txt        # 使用说明文件

二、如何上传到GitHub Pages

1. 在GitHub上创建一个新的仓库
2. 将所有文件上传到该仓库
3. 在仓库设置中启用GitHub Pages（Settings -> Pages -> Source -> 选择main分支 -> Save）
4. 几分钟后，您的网站将可以通过GitHub Pages提供的URL访问

三、如何修改内容

1. 修改公司信息
   - 打开index.html文件
   - 找到需要修改的内容（如公司名称、地址、电话等）
   - 替换为您自己的信息

2. 修改导航栏
   - 导航栏链接已经设置好，指向对应的页面区域
   - 如果需要添加新的导航项，可以在nav-links列表中添加新的li元素

3. 修改宣传视频
   - 找到index.html中的视频部分（id="video"）
   - 将iframe标签中的src属性值替换为您自己的视频嵌入链接
   - 支持B站、视频号、抖音等平台的视频嵌入

4. 修改团队成员信息
   - 找到index.html中的团队部分（id="team"）
   - 替换团队成员的图片、姓名和简介

5. 修改联系表单
   - 联系表单目前是前端展示，没有后端处理
   - 如果需要实际功能，可以集成第三方表单服务或自行开发后端

四、如何添加新的博客文章

1. 复制博客文章模板
   - 复制posts/article-template.html文件
   - 重命名为新的文件名（如article4.html）

2. 修改文章内容
   - 打开新复制的文件
   - 修改文章标题、发布日期和正文内容
   - 可以根据需要添加图片、列表、表格等元素

3. 在首页添加文章链接
   - 打开index.html文件
   - 找到博客部分（id="blog"）
   - 复制一个博客文章卡片的HTML代码
   - 修改标题、日期、摘要和链接地址
   - 如果添加了新的文章，可能需要调整布局或分页

五、如何自定义样式

1. 修改颜色
   - 打开css/style.css文件
   - 找到:root部分，修改颜色变量的值
   - 主色调：--primary-color（赭石色）
   - 辅助色：--secondary-color（浅墨绿）
   - 底色：--background-color（米白色）

2. 修改字体
   - 目前使用的是Google Fonts中的Noto Sans SC
   - 可以在head部分替换为其他字体

六、注意事项

1. 所有图片目前使用的是外部链接，如果需要使用自己的图片，请将图片上传到images文件夹，并修改对应的img标签的src属性
2. 视频使用的是外链嵌入，避免占用GitHub仓库空间
3. 网站设计响应式，适配电脑和手机端
4. 博客功能采用手动更新方式，需要手动添加新文章并更新首页链接

如有任何问题，请随时联系我们！
